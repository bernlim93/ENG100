The project is a unique android application that is a mashup of 4 APIs: Twitter, Facebook , Google Maps and Eventbrite.The basic premise of the application is to search for the most (relevant) tweeted (#hastages) events (--MENTION hastags are matches with an eventbrite database?) in the last --ENTER TIME--- and to pinpoint the location of the said event on Google Maps. The Google Maps interface also allows the users to find out what route to take to go to that event. On the User Interface once a user selects an event that pops up on the map (all events on the map were tweeted for at least --ENTER AMOUNT-- times) , the icon on the map changes and a new sceeen comes up that lists how many people tweeted it, how many facebook friends are attending that event and a summary of the event along with timings plus a link to the event website. The user can also manually search for a particular event r detemine the radius of search for popular events/events with friends. The user also has the option of tweeting about the event himself if he chooses to attend it. This application thus is helpful for social gatherings and finding out the most popular events in a particular area.
